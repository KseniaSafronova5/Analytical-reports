#  Аналитика без аналитика: Автоматизированный отчёт по продажам

**Live demo:** [analytika.ksautomationai.com](https://analytika.ksautomationai.com/)

Интерактивное веб-приложение для мгновенного анализа коммерческих показателей из CSV-файлов. Реализует полный цикл обработки данных: от визуализации аномалий до автоматической отправки ключевых метрик в Telegram через n8n.

> Цель: инструмент для принятия решений «в один клик» — без ручного сбора отчётов.

---

##  Ключевые возможности

- **Мгновенные метрики** — автоматический расчёт выручки, среднего чека и маржинальности
- **Умный анализ** — визуализация продаж по дням недели и автоматическое выявление аномалий (например, критическое падение в выходные)
- **Интеграция с Telegram** — отправка сводки в бот одним нажатием кнопки через n8n webhook
- **Масштабируемость** — гибкая настройка списка получателей (директор, отдел, аналитик)

---

##  Безопасность

- **Синтетические данные** — все демо-файлы содержат только сгенерированные данные
- **HTTPS** — передача данных в n8n по защищённому протоколу
- **In-Memory** — данные обрабатываются в RAM; при закрытии вкладки удаляются
- **Без БД** — файлы не сохраняются на диске

##  Как запустить локально

```bash
# 1. Клонировать репозиторий
git clone https://github.com/KseniaSafronova5/Streamlit.git
cd Streamlit

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Настроить Webhook URL в коде для связи с n8n

# 4. Запустить приложение
streamlit run app.py
```
