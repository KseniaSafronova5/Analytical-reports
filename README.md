Аналитика без аналитика: Автоматизированный отчет по продажам
Описание проекта
Интерактивное веб-приложение на базе Streamlit, предназначенное для мгновенного анализа коммерческих показателей из CSV-файлов. Проект реализует полный цикл обработки данных: от визуализации аномалий до автоматической отправки ключевых метрик в Telegram через интеграционную платформу n8n.
Цель проекта: Предоставить владельцам бизнеса и менеджерам инструмент для принятия решений "в один клик", исключив рутинный ручной сбор отчетов и пересылку скриншотов.
🚀 Ключевые возможности
Мгновенные метрики: Автоматический расчет выручки, среднего чека и маржинальности.
Умный анализ: Визуализация продаж по дням недели и автоматическое выявление аномалий (например, критическое падение продаж в выходные).
Интеграция с мессенджером: Отправка сводки в Telegram-бот одним нажатием кнопки.
Масштабируемость: Архитектура позволяет настраивать список получателей (директор, чат отдела, аналитик) и выбирать адресата прямо в интерфейсе.
🔒 Безопасность и конфиденциальность
Данный проект разработан с приоритетом на защиту корпоративных данных:
Синтетические данные: Все демонстрационные файлы содержат исключительно сгенерированные (синтетические) данные.
Защищенный канал: Передача данных в n8n осуществляется по защищенному протоколу HTTPS.
Принцип "In-Memory": Обработка данных происходит в оперативной памяти сервера. Как только вы закрываете вкладку браузера, все загруженные данные безвозвратно удаляются.
Отсутствие БД: Проект не сохраняет ваши файлы на диске или в базе данных, что исключает риск утечки в открытый доступ.
🛠 Стек технологий
Python 3.x
Streamlit — фронтенд и интерфейс.
Pandas — обработка и анализ данных.
n8n — автоматизация рабочих процессов (Workflow Automation).
Telegram Bot API — доставка уведомлений.
Requests — взаимодействие с вебхуками.
📋 Как запустить
Клонируйте репозиторий.
Установите зависимости: pip install -r requirements.txt.
Запустите приложение: streamlit run your_app.py.
Настройте свой Webhook URL в коде для связи с n8n.
