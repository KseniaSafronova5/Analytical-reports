# Аналитика без аналитики: Автоматизированный отчет по продажам

## Описание проекта
Интерактивное веб-приложение на базе **Streamlit**, предназначенное для мгновенного анализа коммерческих показателей из CSV-файлов. 

Проект реализует полный цикл обработки данных: от визуализации аномалий до автоматической отправки ключевых метрик в Telegram через интеграционную платформу **n8n**. Цель проекта — предоставить владельцам бизнеса и менеджерам инструмент для принятия решений "в один клик", исключив рутинный ручной сбор отчетов.

## Ключевые возможности
* **Мгновенные метрики:** Автоматический расчет выручки, среднего чека и маржинальности.
* **Умный анализ:** Визуализация продаж по дням недели и автоматическое выявление аномалий (например, критическое падение продаж в выходные).
* **Интеграция с мессенджером:** Отправка сводки в Telegram-бот одним нажатием кнопки.
* **Масштабируемость:** Архитектура позволяет настраивать список получателей (директор, чат отдела, аналитик).

## Безопасность и конфиденциальность
* **Синтетические данные:** Все демонстрационные файлы содержат исключительно сгенерированные данные.
* **Защищенный канал:** Передача данных в n8n осуществляется по протоколу HTTPS.
* **Принцип "In-Memory":** Обработка данных происходит в оперативной памяти сервера. Как только вы закрываете вкладку браузера, все загруженные данные удаляются.
* **Отсутствие БД:** Проект не сохраняет ваши файлы на диске.

## Стек технологий
* **Python 3.x / Streamlit:** Фронтенд и интерфейс.
* **Pandas:** Обработка и анализ данных.
* **n8n:** Автоматизация рабочих процессов (Workflow Automation).
* **Telegram Bot API:** Доставка уведомлений.
* **Requests:** Взаимодействие с вебхуками.

## Как запустить
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите приложение: `streamlit run app.py`
4. Настройте свой Webhook URL в коде для связи с n8n.

## Инструкция по настройке Telegram-бота
Чтобы получать уведомления, нужно создать своего бота и «подружить» его с n8n:

Создание бота:

Откройте Telegram и найди пользователя @BotFather.

Отправьте команду /newbot.

Придумайте имя для бота (например, SalesAnalyticsBot).

После создания он пришлет вам API Token (длинная строка с цифрами и буквами). Сохраните её!

Настройка в n8n:

В интерфейсе n8n добавьте узел Telegram.

В разделе Credentials выберите Create new.

В поле Access Token вставьте тот самый токен, который выдал BotFather.

Нажмите Save. Теперь n8n может управлять вашим ботом.

Получение Chat ID:

Чтобы бот знал, куда слать сообщения, нужно найти свой ID (или ID чата группы).

Напишите любое сообщение своему новому боту.

Перейдите по ссылке: https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates (замените <ВАШ_ТОКЕН> на реальный токен).

В ответе найдите поле "chat": {"id": 123456789, ...}. Ваши цифры — это и есть Chat ID.

Запуск:

В узле Telegram в n8n в поле Chat ID вставьте полученные цифры.

Теперь при каждом срабатывании вебхука бот будет присылать сообщение в личку.
